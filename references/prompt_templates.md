## 概览
本文档提供视频分析与风格复刻 Skill 的提示词生成模板和使用指南。提示词用于指导 AI 视频生成系统或人工视频编辑，确保输出符合预期风格和内容要求。

## 提示词类型

### 1. 风格提示词
**用途**: 描述视频的视觉风格、情感基调和技术特征
**应用**: 视频风格化渲染、AI视频生成、风格迁移

### 2. 内容提示词
**用途**: 描述视频的具体内容、场景、人物、叙事元素
**应用**: 视频内容创作、场景生成、情节设计

## 风格提示词模板

### 基础模板结构

```
[风格类别]

视觉特征:
- 色彩方案: [主色调/色彩情感/饱和度]
- 光影效果: [光照类型/阴影处理/高光表现]
- 构图特点: [镜头角度/画面布局/焦点设置]

技术特征:
- 运镜方式: [推/拉/摇/移/跟/升降]
- 剪辑节奏: [快/慢/中速]
- 转场效果: [硬切/淡入淡出/叠化/划像]
- 景深效果: [浅景深/深景深]

情感氛围:
- 基调: [主要情绪词汇]
- 氛围: [环境感觉描述]
- 节奏: [节奏类型描述]
```

## 生成最佳实践

### 1. 结构化生成
- **使用模板**: 基于参考模板生成，确保结构完整
- **分类清晰**: 明确区分风格和内容提示词
- **层次分明**: 按照视觉-技术-情感的层次组织

### 2. 可复用设计
- **参数化**: 使用可替换的参数，支持批量生成
- **模板化**: 创建团队共享的提示词模板库
- **版本管理**: 记录提示词版本，便于追溯和优化

### 3. 精准表达
- **具体描述**: 使用具体的词汇，避免模糊表达
- **可操作**: 提示词应该能够被理解和执行
- **可度量**: 尽可能使用可量化的描述

### 4. 迭代优化
- **反馈收集**: 收集使用反馈，持续优化提示词
- **A/B测试**: 对比不同提示词的效果
- **机器学习**: 利用AI辅助优化提示词质量
