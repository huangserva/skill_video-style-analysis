# 视频分析与风格复刻 - 提示词示例

## 示例1：电影风格Vlog

### 风格提示词

```
电影风格 (Cinematic)

视觉特征:
- 色彩方案: 暖色调、电影感调色、略带胶片质感，偏暖黄和橙色
- 光影效果: 戏剧性光效、明暗对比强烈、柔和边缘阴影、黄昏金色光晕
- 构图特点: 黄金分割构图、背景虚化、中心聚焦、前景虚化

技术特征:
- 运镜方式: 缓慢推镜头、稳定器拍摄、细微摇镜、跟随主体
- 剪辑节奏: 中速偏慢、流畅转场、避免快速跳切
- 转场效果: 淡入淡出、叠化、柔和过渡
- 景深效果: 浅景深、主体突出、背景模糊

情感氛围:
- 基调: 浪漫、温情、怀旧、诗意
- 氛围: 沉浸式、电影感、艺术性、温暖治愈
- 节奏: 优雅、舒缓、有韵律、缓慢而有力
```

### 内容提示词

```
人物故事内容 - 旅行回忆

场景描述:
- 环境: 欧洲小镇、黄昏时分、石板路、咖啡店露台
- 人物: 主角1人，20-30岁，休闲服装，手持咖啡，微笑表情
- 道具: 摄影相机、旅行背包、咖啡杯、笔记本

叙事元素:
- 主题: 旅行中的美好回忆、生活感悟、时光流逝
- 情节: 入场-探索咖啡馆-与人交谈-独处思考-感悟结尾
- 冲突: 独处与社交的内心平衡、时光流逝的感慨

视听要素:
- 画面: 咖啡热气特写、主角微笑表情、小镇全景、夕阳镜头
- 声音: 轻柔钢琴背景音乐、环境音效(咖啡机、街道声音)、内心独白
- 文字: 标题"巴黎的黄昏"、章节划分、情感字幕
```

## 示例2：纪录片风格自然风光

### 风格提示词

```
纪录片风格 (Documentary)

视觉特征:
- 色彩方案: 自然色彩、高对比度、真实还原、饱和度适中
- 光影效果: 自然光、明暗平衡、细节丰富、晨昏光效
- 构图特点: 纪实构图、观察者视角、多角度拍摄、全景与特写结合

技术特征:
- 运镜方式: 手持拍摄、跟拍、固定机位、缓慢摇镜
- 剪辑节奏: 节奏多变、跟随内容、动态调整、长镜头
- 转场效果: 硬切、直接切换、自然过渡
- 景深效果: 深景深、全景清晰、层次分明

情感氛围:
- 基调: 客观、真实、有温度、自然
- 氛围: 沉浸、真实、有力量、宁静
- 节奏: 流畅、自然、有机、呼吸感
```

### 内容提示词

```
自然风光内容 - 森林日出

场景描述:
- 环境: 原始森林、清晨薄雾、阳光穿透、露水珠
- 人物: 无（纯自然风光）
- 道具: 摄影设备、三脚架、自然环境元素

叙事元素:
- 主题: 自然之美、环境保护、生命力量、时间流逝
- 情节: 黎明前的黑暗-第一缕阳光-森林苏醒-生命活跃
- 冲突: 黑暗与光明的交替、生命的脆弱与坚韧

视听要素:
- 画面: 广角全景、光线穿透树林、露水特写、鸟类飞翔
- 声音: 鸟鸣声、风声、自然环境音、轻柔背景音乐
- 文字: 地点名称"亚马逊雨林"、解说文字、时间标注
```

## 示例3：商业广告风格产品展示

### 风格提示词

```
商业广告风格 (Commercial)

视觉特征:
- 色彩方案: 品牌色彩(蓝色系)、高对比、鲜艳醒目、专业质感
- 光影效果: 专业布光、精细控制、完美质感、产品高光
- 构图特点: 产品中心、品牌突出、视觉冲击、简洁有力

技术特征:
- 运镜方式: 精准运镜、多角度展示、产品特写、流畅旋转
- 剪辑节奏: 精准控制、配合音乐、节奏感强、快速有力
- 转场效果: 创意转场、特效叠加、品牌元素过渡
- 景深效果: 浅景深、焦点变化、产品清晰、背景简洁

情感氛围:
- 基调: 专业、高端、有吸引力、科技感
- 氛围: 精致、高品质、令人向往、未来感
- 节奏: 有力、精准、有冲击力、充满活力
```

### 内容提示词

```
产品展示内容 - 智能手机广告

场景描述:
- 环境: 现代简约工作室、纯净白色背景、专业灯光
- 人物: 模特1人，时尚着装，手持产品，自信表情
- 道具: 智能手机、展示台、生活场景道具

叙事元素:
- 主题: 产品优势、使用价值、品牌理念、科技改变生活
- 情节: 产品展示-功能演示-使用场景-品牌印象
- 冲突: 用户的痛点(复杂难用) vs 产品的解决方案(简单智能)

视听要素:
- 画面: 产品特写(细节质感)、屏幕显示、使用场景、品牌标识
- 声音: 科技感背景音乐、产品音效(点击、滑动)、专业解说词
- 文字: 产品名称、核心卖点(快速、智能、高清)、品牌标识
```

## 示例4：Vlog风格日常生活

### 风格提示词

```
Vlog风格

视觉特征:
- 色彩方案: 明亮色彩、高饱和度、清新自然、阳光感
- 光影效果: 自然明亮、均匀光照、柔和阴影、日光效果
- 构图特点: 居中构图、自拍视角、第一人称、随意自然

技术特征:
- 运镜方式: 快速切换、手持跟拍、多角度、自拍杆
- 剪辑节奏: 快速、活泼、动感、跳跃
- 转场效果: 快速转场、动感特效、自然切换
- 景深效果: 浅景深、背景虚化、主体突出

情感氛围:
- 基调: 活力、友好、亲切、快乐
- 氛围: 轻松、有趣、有感染力、积极向上
- 节奏: 轻快、跳跃、有节奏、充满能量
```

### 内容提示词

```
人物故事内容 - 美食制作Vlog

场景描述:
- 环境: 现代厨房、明亮的自然光、整洁的台面
- 人物: 主角1人，休闲着装，围裙，微笑表情
- 道具: 厨房用具、食材、成品、装饰

叙事元素:
- 主题: 美食制作、生活乐趣、分享快乐、技能展示
- 情节: 准备食材-制作过程-成品展示-品尝分享
- 冲突: 制作过程中的小插曲 vs 最终成功的喜悦

视听要素:
- 画面: 食材特写、制作过程、成品展示、表情反应
- 声音: 轻快背景音乐、制作音效(切菜、搅拌)、解说词
- 文字: 菜谱名称、步骤标注、小贴士、结尾感谢
```

## 提示词使用建议

### 1. 风格提示词使用
- **直接引用**: 可以直接复制使用完整风格提示词
- **参数调整**: 根据具体需求调整颜色、光线、节奏等参数
- **混合使用**: 可以混合不同风格的元素创建独特风格

### 2. 内容提示词使用
- **模板填充**: 根据具体内容填充模板中的空白部分
- **场景定制**: 根据实际场景调整环境、人物、道具描述
- **叙事调整**: 根据故事需要调整情节和冲突描述

### 3. 提示词优化
- **迭代完善**: 根据生成结果不断优化提示词
- **A/B测试**: 对比不同提示词的效果
- **反馈收集**: 收集用户反馈，持续改进

### 4. 团队协作
- **标准化**: 建立团队统一的提示词标准和格式
- **共享库**: 建立团队提示词共享库，提高效率
- **版本管理**: 记录提示词版本，便于追溯和复用
